SEEFA - Secure Key Manager

This is a React Native application built with Expo.

How to Run This App:

  • 1. Ensure you have Node.js and npm/yarn installed.
  • 2. Install Expo CLI: npm install -g expo-cli
  • 3. Navigate to the project directory in your terminal.
  • 4. Install dependencies: npm install (or yarn install)
  • 5. Start the development server: npx expo start
  • 6. Follow the instructions in the terminal to open the app on a physical device (using the Expo Go app) or an emulator/simulator.

For more information, visit the Expo Documentation.
